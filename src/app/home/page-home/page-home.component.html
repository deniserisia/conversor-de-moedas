Bem-vindo à Coins

Coins é o seu conversor ideal de moedas.

Denise Rísia Carvalho

Este projeto de conversor de moedas foi um desafio final de nivelamento do projeto de Residência de Software da Universidade Federal do Estado do Rio de Janeiro.

O desafio tinha como objetivo o desenvolvimento de uma aplicação em Angular que realize a conversão de valores monetários entre diferentes moedas. A aplicação deve consumir o serviço de conversão de moedas exchangerate do seguinte site: https://exchangerate.host/
